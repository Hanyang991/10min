clock_game/

## 실행 방법

### 1. 백엔드 (Python)
```bash
pip install flask
python app.py
```
> `http://localhost:5000` 에서 백엔드 시작

### 2. 프론트엔드
`index.html` 을 **Live Server** (VS Code 확장) 등으로 열거나,
단순히 이중클릭해도 됩니다.

> ⚠️ `file://` 프로토콜로 열면 CORS 차단될 수 있으므로 **Live Server 사용 권장** (예: `http://127.0.0.1:5500/index.html`)

## 게임 규칙
1. 스테이지마다 랜덤 조건이 주어진다
2. 조건이 충족되는 순간 **스페이스바**를 누르세요
3. 조건 미충족 시 실패 → 같은 스테이지 재도전
4. 10초 내에 멈추지 못하면 자동 실패
5. 최대 스테이지 기록이 순위榜에 저장된다

### 난이도 증가
- **1~4 스테이지**: 시계 1개
- **5~9 스테이지**: 시계 2개
- **10~14 스테이지**: 시계 3개
- **15~19 스테이지**: 시계 4개
- **20+ 스테이지**: 시계 5개 (최대)

시계가 많아질수록 정답을 찾기 어려워집니다!

### 조작
- **스페이스바**: 
  - STOP (조건 충족 시)
  - 결과 화면에서 다음 스테이지/재시도
- **마우스 클릭**: STOP 버튼 직접 클릭 가능

## 조건 종류 (랜덤)
| 유형 | 설명 | 비고 |
|------|------|------|
| 특정 숫자 | 시·분·초 중 특정 숫자가 표시될 때 | 숫자 표시 시계에만 |
| 연속 숫자 | 같은 숫자가 N개 연속으로 나타날 때 | 숫자 표시 시계에만 |
| 회문 | 시간 숫자열이 앞뒤로 같아질 때 | 숫자 표시 시계에만 |
| 숫자 포함 | 특정 숫자(0~9)가 시간에 포함될 때 | 숫자 표시 시계에만 |
| 숫자 미포함 | 특정 숫자(0~9)가 시간에 없을 때 | 숫자 표시 시계에만 |
| 숫자 합 | 숫자들의 합이 목표 값을 달성할 때 | 숫자 표시 시계에만 |
| 배경색 변화 | 배경이 특정 색으로 바뀌면 | 해당 조건일 때만 배경 변화 |
| 아이콘 등장 | 특정 이모지가 화면에 나타나면 | 해당 조건일 때만 아이콘 등장 |
| 시계 강조 | 특정 종류의 시계가 빛나는 순간 | |
| 초 = 00 | 시계의 초가 00이 될 때 | 현재 초 50+ 일 때만 |
| 스페이스바 N번 | 스페이스바를 정확히 N번 누르기 | 3~7회, 10초 후 자동 검증 |

**중요**: 아날로그 시계만 있을 때는 숫자 관련 조건이 나오지 않습니다!

## 시계 종류
- **디지털** – 숫자 표시 (HH:MM:SS)
- **아날로그** – 바늘 시계 (SVG)
- **바이너리** – BCD 이진 표시 (각 열은 시·분·초의 십의 자리/일의 자리, 켜진 비트 = 1)
- **플립** – 플립카드 스타일
- **네온** – 네온 글로우 효과

## 백엔드 API
| Endpoint | Method | 설명 |
|----------|--------|------|
| `/api/register` | POST | 플레이어 등록 |
| `/api/new_event` | GET | 스테이지별 랜덤 이벤트 생성 |
| `/api/verify` | POST | 멈추기 조건 검증 |
| `/api/save_record` | POST | 기록 저장 |
| `/api/leaderboard` | GET | 순위 목록 |
| `/api/my_best` | GET | 개인 최고기록 |

---
기록은 `game.db` (SQLite)에 저장됩니다.
